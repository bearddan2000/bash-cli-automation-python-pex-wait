# bash-cli-automation-python-pex-wait

## Description
Conversion of python normal build with docker wait
to pex stand alone.

## Tech stack
- bash
- python
    - pyaml

## Note
<ol>
    <li>Its assumed that docker wait is setup.</li>
    <li>To run this script you'll need python and pyaml</li>
</ol>

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
